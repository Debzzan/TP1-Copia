\doxysubsection{Detailed Description}
Classe que implementa a interface de serviço de hospedagem. 

Esta classe herda de Interface\+Servico\+Hospedagem e \doxylink{class_modelo}{Modelo}, e implementa os métodos para criar, ler, atualizar e remover hospedagens.
