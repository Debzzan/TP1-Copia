\doxysubsection{Detailed Description}
Classe que implementa a interface de serviço de destino. 

Esta classe herda de Interface\+Servico\+Destino e \doxylink{class_modelo}{Modelo}, e implementa os métodos para criar, ler, atualizar e remover destinos.
