\doxysubsection{Detailed Description}
Classe que implementa a interface de serviço de atividade. 

Esta classe herda de Interface\+Servico\+Atividade e \doxylink{class_modelo}{Modelo}, e implementa os métodos para criar, ler, atualizar e remover atividades.
