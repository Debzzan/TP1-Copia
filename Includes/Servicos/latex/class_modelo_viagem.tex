\doxysubsection{Detailed Description}
Classe que implementa a interface de serviço de viagem. 

Esta classe herda de Interface\+Servico\+Viagem e \doxylink{class_modelo}{Modelo}, e implementa os métodos para criar, ler, atualizar e remover viagens.
